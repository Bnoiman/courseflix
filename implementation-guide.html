CourseFlixAI Implementation Guide

This guide will help you implement the AI chat feature with mock data on your CourseFlixAI website. The implementation uses client-side JavaScript and doesn't require any paid APIs or backend services.

Files Overview

I've prepared the following files for you to add to your repository:

  • data/courses.json - Mock course data and recommendations
  • js/ai-recommendations.js - AI conversation logic with mock responses
  • js/chat.js - Chat interface implementation
  • css/chat-styles.css - Styling for the chat interface

Implementation Steps

1. Create Directory Structure

First, create the necessary directories in your repository:

mkdir -p data
mkdir -p js
mkdir -p css
mkdir -p images/courses

2. Add Course Data

data/courses.json

This file contains mock course data and recommendation lists. Add it to your repository's data directory.

3. Add JavaScript Files

js/ai-recommendations.js

This file contains the mock AI conversation logic. Add it to your repository's js directory.

js/chat.js

This file implements the chat interface. Add it to your repository's js directory.

4. Add CSS Styles

css/chat-styles.css

This file contains the styling for the chat interface. Add it to your repository's css directory.

5. Update Your HTML

Add the following lines to your index.html file, just before the closing </body> tag:

<!-- AI Chat Interface -->
<div class="chat-button">
  <svg width="24" height="24" viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg">
    <path d="M20 2H4C2.9 2 2 2.9 2 4V22L6 18H20C21.1 18 22 17.1 22 16V4C22 2.9 21.1 2 20 2Z" fill="white"/>
  </svg>
</div>

<!-- JavaScript Files -->
<script src="js/ai-recommendations.js"></script>
<script src="js/chat.js"></script>

Also add this line in the <head> section of your HTML:

<link rel="stylesheet" href="css/chat-styles.css">

6. Add Course Images (Optional)

For the best experience, add course thumbnail images to the images/courses/ directory. The image filenames should match the course IDs in the JSON file, for example:

  • images/courses/ml-az.jpg
  • images/courses/python-crash.jpg
  • images/courses/web-dev-bootcamp.jpg

If you don't have images, you can also add a fallback image:

images/courses/course-placeholder.jpg

Note: If you don't have course images, the chat interface will still work, but it will show broken image links. You can modify the chat.js file to use a default image instead.

Testing Your Implementation

After adding all the files and updating your HTML:

  1. Commit and push the changes to your GitHub repository
  2. Visit your site at https://bnoiman.github.io/courseflix/
  3. You should see a chat button in the bottom right corner
  4. Click the button to open the chat interface
  5. Try asking about different topics like "Python", "Web Development", or "Data Science"

Troubleshooting

Chat Button Not Appearing

  • Check that all JavaScript and CSS files are correctly linked in your HTML
  • Check the browser console for any errors
  • Ensure the files are in the correct directories

Course Images Not Loading

  • Verify that your image paths are correct
  • Add a course-placeholder.jpg image as a fallback
  • Check that the image filenames match the course IDs in the JSON file

Chat Not Responding

  • Check that the courses.json file is being loaded correctly
  • Verify that the ai-recommendations.js file is properly linked
  • Look for JavaScript errors in the browser console

Important: This implementation uses client-side JavaScript and mock data. For a production environment, you would want to implement server-side processing for the AI recommendations and real course data.
